# NAGOYAMESHI（ナゴヤメシ）

名古屋のB級グルメに特化した  
**店舗検索・レビュー・予約機能を備えたWebアプリケーション**です。

Djangoを用いたWebアプリ開発の学習として、  
**実務を意識した設計・機能分離・セキュリティ管理**を重視して制作しています。

---

## 制作目的
- SAMURAI ENGINEER カリキュラム修了後の課題として制作
- Djangoを用いたWebアプリ開発の一連の流れを理解・実装する
- 会員側／管理者側を分離した実務的な構成を学ぶ
- GitHubに公開可能な、セキュリティを意識した構成を作る

---

## アプリ概要
NAGOYAMESHIは、名古屋のB級グルメ店を  
**「探す・予約する・レビューする」** ことを目的としたWebアプリです。

ユーザー向け機能と管理者向け機能を分けて実装しています。

---

## 主な機能
※一部機能は設計済み・今後実装予定です

### 会員側
- 店舗一覧／詳細表示
- 会員登録／ログイン
- マイページ
  - プロフィール編集
  - メールアドレス変更
  - パスワード変更
- 有料会員への移行 (予定)
- カテゴリ別店舗検索（予定）
- お気に入り登録（有料会員／予定）
- レビュー投稿（有料会員／予定）
- 店舗予約（有料会員／予定）

### 管理者側
- 管理者ログイン
- 店舗情報のCRUD管理
- 会員管理
- カテゴリ管理
- 会社情報・利用規約管理

